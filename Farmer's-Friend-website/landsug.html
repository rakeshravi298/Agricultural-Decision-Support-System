Land suggestion dashboard

This dashboard can be used to select the district to buy the agricultural land according to the state you want and the crops you want to grow.

Please select the State you want to buy a land from the given filter drop down. Also select the Crops that you want to plant in the land from the filter dropdown menu. Ensure that you give all the values correctly.

Data has been obtained for each and every district and we have the crop's yearly historic data.

In the bar graph the state's districts are in the rows and the average of the crop's yields are calculated and represent the length of the bars. Higher the average yield , more better it is for buying the agricultural land. The state's average yield for the set of crops you have given is also summarised , which can be used for comparison with the district's yield averages. The Pie chart gives an insight about the percentage of the area for the set of crops given when compared to the entire state's area. Higher the percentage , higher number of farmers are planting those crops in that district which means there may be a proper support/infrastructure for the given crop.
